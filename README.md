# 链式哈希表性能实验配套源码
数据结构课程学术海报完整工程，基于 Python 实现自动扩容链式哈希表，完成哈希查询性能梯度测试、TwoSum 双算法性能对照，自动生成可视化图表。

## 一、项目功能
1. 自定义链式地址哈希表，设置负载因子0.75，达到阈值自动扩容重哈希
2. 哈希查询耗时梯度测试，生成数据规模-查询耗时折线图
3. TwoSum 暴力枚举解法、哈希表优化解法性能对比
4. 一键批量执行全部性能测试与绘图，图表自动保存至结果目录
5. 支持外部自定义测试数据集导入

## 二、项目目录结构
```
hashmap_poster/
├── data/ # 输入测试数据集（支持 txt、json 格式）
├── src/
│ ├── hash_table.py # 哈希表核心实现：插入、查询、扩容、删除
│ ├── two_sum.py # TwoSum 暴力算法 & 哈希优化算法
│ ├── performance_test.py # 批量性能测试、计时统计逻辑
│ ├── draw_perf.py # matplotlib 可视化绘图代码
│ └── result/ # 程序输出目录：性能对比 PNG 图表
├── run_all.py # 一键运行全部测试与绘图入口脚本
├── LICENSE # MIT 开源许可证
└── README.md # 项目说明文档
```
## 三、运行环境要求
Python 3.10 及以上版本  
第三方依赖库：matplotlib

## 四、完整使用教程
1. 自定义测试数据（可选操作）
将批量测试用例文件放入 `data/` 文件夹，程序内置读取逻辑支持 `.txt`、`.json` 两种格式。

2. 一键执行全部实验
在项目根目录打开终端，运行入口脚本：
```bash
python run_all.py
```

3. 查看实验输出结果
代码运行完毕后，两张性能可视化对比图表会自动生成并保存到 `src/result/` 文件夹内。
